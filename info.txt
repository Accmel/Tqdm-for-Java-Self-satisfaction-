Information of this library "Tqdm".

How to use-----
  You can use this library to visualize the for-loop and while-loop progress.

  Please import boot.Tqdm .
  Make Instance of Tqdm.
    (The argument for constructor is All-Loop-Num.)
    !!Please make instance "just before" loop!!
  Call Tqdm.tqdm-Method for each loop.

Example programing-----
